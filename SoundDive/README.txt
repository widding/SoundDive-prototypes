CONTROLS:	Kinect_1_traces.pde
-----

key '1'		toggle between mouse and Kinect control
key '2'		toggle displaying the perimeter of the sketch
key '3'		toggle easing on X/Y coordinates. 
		then use "easingX, easingY" instead of "controllerX, controllerY".
key '4'		toggle auto cleaning of the sketch on/off. 
		timer can be set in global variables.
key '5'		toggle displaying the target of the easing controller
